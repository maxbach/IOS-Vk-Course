----- Идеи: -------------

1. Приложения для удобного чтения pdf книг. 

- Вместо использования алгоритмов распознавания текста, приложение будет нарезать файл на строчки и выводить их в презентабельном виде.
- Добавить распознавание картинок, диаграмм
- Функции: сохранение прогресса чтения

2. Агрегатор новостей из соцсетей

- Программа собирает новости из различных соцсетей, сайтов
- Возможность распределения источников по секциям/разделам
- Существуют заранее заготовленные шаблоны разделов. Пользователь может выбрать их сам

3. Приложение, которое содержит настольные игры

- Большинство настольных игр не требует игровых полей, особых фигурок и других сложных изделий. В основном, нужны карточки, какое-то игровое поле, несложное
- Идея состоит в том, чтобы игроки соединялись по Bletooth, чтобы программа уже распределяла материал между игроками
- Например, для игры в мафию нужны карточки. Игроки соединяются друг с другом, программа рассылает карточки игрокам, заменяет ведущего. И таких примеров, сотни
- Можно создать базу данных таких игр для компании.
